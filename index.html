Color Flipper 🐬

Color Flipper es una aplicación realizada con HTML5, CSS3 y JS hecha para quienes necesiten maquetar una página web, o tener una guía de como hacerlo. Consta de tres "partes", una Navbar, un cuerpo principal que llamamos Main y un pie de página llamado Footer. Para interactuar con la aplicación sólo se debe elegir un color y que parte se desea cambiar.
